# Sass Bootstrap 3.3.4 JS test
Just a quick repo testing out Sass-Bootstrap 3.3.4 for an issue. This project uses [Grunt](http://gruntjs.com/). Sass BS is pulled in via Bower.

## Setup
`bower install` then `npm-install`

## Run the app
``grunt serve``

## Folder Structure
/scripts/ = main.js<br>
/styles/ = main.scss
